# Design

This module contains design specefic resources such as 
* colors
* dimens
* styles
* themes
* drawables

all common design elements should be put in this module.

## Dependencies

The module transitively exposes following dependencies
* `androidx.appcompat:appcompat`
* `com.google.android.material:material`

The module itself should not depend on any other module in the project.
